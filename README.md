# flutter_rust

A demo project showcasing bridge between flutter and rust


## Run the project

Prerequisites
Before you begin, ensure you have the following installed:


Rust should installed on the system using rustup

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Flutter SDK:

`https://docs.flutter.dev/install`

## Installation

Clone the repository
```bash
git clone https://github.com/avikarefin/flutter_rust.git
cd flutter_rust
```

Install dependencies

```bash
flutter pub get
```

Launch the app
```bash
flutter run
```
